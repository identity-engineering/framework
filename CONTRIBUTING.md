# Contributing to Identity Engineering

Thank you for your interest in contributing to Identity Engineering!

We are building this as an **open, adoptable framework**. Contributions of all kinds are welcome — from feedback and ideas to content, examples, and code.

## How to Contribute

### 1. Feedback & Ideas
- Open an issue with the label `feedback` or `idea`
- Join the discussion on how to improve the framework

### 2. Content & Documentation
- Improve explanations in `/content`
- Add practical examples
- Help with the mapping to existing theories (Wissner-Gross, Friston, etc.)

### 3. Code & Website
- Work on the website in `/website`
- Improve structure, performance, or accessibility

### 4. Reporting Issues
Please use the issue templates when available.

## Development Setup

```bash
git clone https://github.com/identity-engineering/framework.git
cd framework
git checkout initial-setup
```

## Code of Conduct
We follow a simple principle: be respectful, curious, and focused on building something useful and open.
